### Stress 

A simple yet powerfull HTTP Load Tester.

#### Usage

```
stress -u https://beyondthesyntax.substack.com -n 100000 -c 30
```

| Parameter                      | Usage |
|--------------------------------|-------|
| -u | URL to stress test                |
| -n | Number of requests to make        |
| -c | Number of concurrent requests     |
